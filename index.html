About Me

I started coding in college but the code world really spoke to me at my first workplace. Our company's method to create new employees in Active Directory started my journey. We manually entered each one, so I started thinking 'There has to be a better way'. I was determined to find a better solution. I learned about C# and the .Net framework and eventually created a console application to do the process for me. At first, it was supposed to only be a console application but it turned into a internal web application for our HR department. Now, I am here at Lambda!
